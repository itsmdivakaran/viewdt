Everything you need to explore data

All features are profiled in Python before the HTML renders — no backend, no CDN, no internet.

📊
Spark Histograms
Inline SVG mini-charts in every column header — numeric, categorical, logical, and datetime columns each get a tailored chart.
⚡
Virtualised Grid
Only visible rows are rendered as position:absolute nodes. Handles hundreds of thousands of rows.
🔎
Visual Query Builder
Multi-condition AND/OR filtering with type-aware operators. Numeric, text, logical, and datetime operators each exposed separately.
📋
Code Export
Generates reproducible pandas, Python, and SQL matching the current filter state and visible columns.
🌓
Dark / Light / Auto Theme
Follows the OS preference by default. Toggle at runtime or force via theme="dark".
🏷
Column Labels
Reads series.attrs["label"] — compatible with haven-imported datasets and ADaM clinical data.
📈
Completeness Bars
Per-column fill-rate indicator below each header — green ≥90%, amber ≥70%, red <70%.
🔍
Data Insights Drawer
Click any column header for a full histogram, descriptive statistics panel, and category frequency chart.
👁
Column Picker
Toggle column visibility at runtime without reloading the widget or re-running the cell.
🌐
Standalone HTML
One self-contained file — no CDN, no network. Share as an email attachment or commit to git.
📓
Jupyter Native
Renders inline in JupyterLab, Jupyter Notebook, VS Code notebooks, and Google Colab via _repr_html_().
🔢
Type Badges
# numeric  ·  A text  ·  T/F logical  ·  ⏱ datetime — visible at a glance in every header.

Quick Start

One import, one function call.

import pandas as pd
from viewdt import viewdt

df = pd.read_csv("data.csv")

viewdt(df)                # renders inline in Jupyter / Colab
viewdt(df).show()         # opens in default browser
viewdt(df).save("explorer.html")  # standalone file
from viewdt import viewdt, viewdt_options

viewdt(
    df,
    options=viewdt_options(
        theme="dark",
        hidden_columns=["internal_id", "created_at"],
        hist_bins=30,
        top_n=15,
        missing_bars=True,
    ),
)
from viewdt import (
    viewdt,
    load_iris, load_mtcars, load_penguins,
    load_tips, load_gapminder, load_titanic, load_stocks,
    list_datasets,
)

viewdt(list_datasets())   # summary table of all datasets
viewdt(load_iris())       # 150 rows — flower measurements
viewdt(load_mtcars())     # 32 rows  — 1974 car specs
viewdt(load_titanic())    # 891 rows — survival + missing values
viewdt(load_stocks())     # 1260 rows — datetime price series
from viewdt import save_viewdt, viewdt_options

# One-line export to a standalone HTML file
save_viewdt(
    df,
    "team_report.html",
    options=viewdt_options(theme="light", code_export=False),
    dataset_name="sales_q1",
)

# Or via the widget
w = viewdt(df)
w.save("explorer.html")   # export file
w.show()                   # open in browser
import pandas as pd
from viewdt import viewdt

# Column labels (compatible with haven / ADaM datasets)
df["AVAL"].attrs["label"] = "Analysis Value"
df["PARAMCD"].attrs["label"] = "Parameter Code"
df["VISITNUM"].attrs["label"] = "Visit Number"

viewdt(df)   # labels appear below column names in headers

API Reference

All viewdt_options() parameters are keyword-only with sensible defaults.

ParameterTypeDefaultDescription
theme"auto" | "light" | "dark""auto"UI colour scheme; auto follows OS preference
show_labelsboolTrueShow series.attrs["label"] in column headers
histogramsboolTrueRender spark histograms / category bars
missing_barsboolTrueShow data-completeness bar in headers
type_badgesboolTrueShow data-type badges in column headers
insightsboolTrueEnable the Data Insights drawer (click any header)
query_builderboolTrueEnable the visual query builder
column_pickerboolTrueEnable the column-visibility picker
code_exportboolTrueEnable reproducible code generator (pandas / Python / SQL)
global_searchboolTrueEnable the global search box
na_stringstr"NA"Text displayed for missing values
hidden_columnslist[str] | NoneNoneColumns hidden on initial render
page_sizeint200Rows kept in the virtualised DOM buffer
hist_binsint20Bin count for numeric / datetime histograms
top_nint10Number of categories profiled per text column
max_cellsint5 000 000Warn when nrow × ncol exceeds this threshold
